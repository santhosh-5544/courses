# 🎓 Courses Dashboard

A modern and interactive **Courses Learning Platform** built using HTML, CSS, and JavaScript.  
This project provides structured learning paths, exercises, and coding resources from **beginner to advanced level**.

---

## 🚀 Features

- 📚 Multiple Courses (Python, Java, HTML, CSS, JavaScript, C++, MySQL, NodeJS, React, MongoDB)
- 🧠 Exercises Section (Basic → Advanced questions)
- 💻 Built-in Compiler Button (UI ready)
- 🎨 Modern UI (Glassmorphism + Gradient Design)
- 📱 Fully Responsive Design
- ⚡ Fast & Lightweight (No frameworks)

---

## 🧩 Pages Included

- `index.html` → Main dashboard  
- `python.html` → Python course content  
- `java.html` → Java course  
- `html.html` → HTML course  
- `css.html` → CSS course  
- `javascript.html` → JavaScript course  
- `cpp.html` → C++ course  
- `mysql.html` → MySQL course  
- `nodejs.html` → NodeJS course  
- `react.html` → React course  
- `mongodb.html` → MongoDB course  
- `exercise.html` → Practice questions (200+)
