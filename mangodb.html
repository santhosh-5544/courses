🍃 MongoDB Roadmap (Basic → Advanced)

1️⃣ Database Fundamentals

MongoDB is a NoSQL database that stores data in flexible JSON-like documents.

SQL Database → Tables MongoDB → Collections Table Row → Document

2️⃣ MongoDB Setup

Download MongoDB Install MongoDB Install MongoDB Compass Run MongoDB Shell: mongosh

3️⃣ MongoDB Basics

Database → myDatabase Collection → users Document Example { name: "John", age: 25, city: "London" }

4️⃣ CRUD Operations

db.users.insertOne({ name:"Alex", age:22 }) db.users.find() db.users.updateOne( {name:"Alex"}, {$set:{age:23}} ) db.users.deleteOne({name:"Alex"})

5️⃣ Querying Data

db.users.find({ age: {$gt:20} }) db.users.find({ $or:[ {age:20}, {age:25} ] })

6️⃣ Sorting & Limiting

db.users.find() .sort({age:1}) .limit(5) db.users.find().skip(5)

7️⃣ Indexing

db.users.createIndex({name:1}) db.users.createIndex( {name:1}, {unique:true} )

8️⃣ Aggregation Framework

db.orders.aggregate([ {$match:{status:"A"}}, {$group:{ _id:"$cust_id", total:{$sum:"$amount"} }} ])

9️⃣ Data Modeling

Embedded Document { name:"John", address:{ city:"London", zip:10001 } }

🔟 Relationships

User Collection { _id:1, name:"John" } Order Collection { user_id:1, product:"Laptop" }

1️⃣1️⃣ MongoDB with Node.js

const {MongoClient} = require("mongodb"); const client = new MongoClient("mongodb://localhost:27017"); async function run(){ await client.connect(); const db = client.db("test"); const users = db.collection("users"); const data = await users.find().toArray(); console.log(data); } run();
